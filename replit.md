# Portfolio Website

React + Vite + TypeScript single-page app (Figma Make export).

## Stack
- Vite 6 + React 18 + TypeScript
- Tailwind v4, Radix UI, MUI, Motion

## Replit Setup
- Workflow `Start application` runs `npm run dev` on port 5000.
- `vite.config.ts` server bound to `0.0.0.0:5000` with `allowedHosts: true` for the Replit proxy.
- Deployment: static, build `npm run build`, publicDir `dist`.
